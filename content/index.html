Slidy

Web framework for authoring and showing presentations

Press → or Space to go to next slide

Made by Valentin Nemcev while applying for a job at Yandex

How it works

  1. Use HTML to write your presentation
  2. Use browser to review and present
  3. Focus on content, not presentation ;)

For web developers, by web developer

image

How to use it

  1. Download presentation template
  2. Open index.html in your favourite HTML editor
  3. Use provided examples and comments as a reference
  4. Optionaly setup browser auto-refresh
    Live Server for Node.js is a good choice

Keyboard controls

Should work with remote presenters

→, Space to go to next slide

← to go to previous slide

Well-behaved urls

Slidy uses hash part (fragment identifier) of the page url to keep track of slideshow state

  • Each slide is directly linkable
    Link to previous slide
  • Position is preserved through browser refresh
  • Back button brings back slide sorter, not previous slide
