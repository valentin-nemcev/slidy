How it works

Use HTML to write your presentation

Use browser to review and present

Focus on content, not presentation ;)

Deck view — slide sorter

All slides at a glance

Use for quick review or to pick specific slide

Slide show — keyboard controls

Control with keyboard

  • →, Space to go to next slide
  • ← to go to previous slide
  • Esc to stop slide show

Design goals

  • Use nested list to structure presentations
    1. Presentation
    2. Section
    3. (Sub-section)
    4. Slide
    5. List item
    6. (Sub-list item)
  • Have an outline view based on nested list structure
  • Perfect UI
  • Have a “slide sorter” view, when multiple slides are displayed on one screen
  • Write presentation as HTML or some other templating language, have live preview in browser, have an opinionated set of CSS and JS defaults, extend it only in special circumstances, start with a template/reference project
  • No presenter view
  • Bonus goal: produce handouts and presenter notes ready for printing
