Slidy

Lightweight framework for authoring and presenting slideshows

Made by Valentin Nemcev while applying for a job at Yandex

How it works

You write your presentation using HTML (or something that compiles to it) following specific structure

You use CSS to style your presentation

You use Slidy JavaScript library to show your presentation yourself slide by slide or embed it in a webpage

Design goals

  • Use nested list to structure presentations
    1. Presentation
    2. Section
    3. (Sub-section)
    4. Slide
    5. List item
    6. (Sub-list item)
  • Have an outline view based on nested list structure
  • Perfect UI
  • Have a “slide sorter” view, when multiple slides are displayed on one screen
  • Write presentation as HTML or some other templating language, have live preview in browser, have an opinionated set of CSS and JS defaults, extend it only in special circumstances, start with a template/reference project
  • No presenter view
  • Bonus goal: produce handouts and presenter notes ready for printing

Two parts

Slidy-presenter — “runtime”, a JS library used for presenting

Slidy-template — author environment, a project template used for authoring
